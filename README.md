## アプリ名
Musicrossing


## 概要

ミュージシャン専用のマッチングサイトです。


<br>

## 使い方  

アカウント作成し、各ユーザーのプロフィール情報から検索と閲覧ができます。<br>
気になった人にはダイレクトメッセージを送ることが出来ます。
 

<br>

## 開発環境
MAMP<br>
MySQL<br>
PHP<br>
HTML<br>
CSS<br>


<br>

## データベース
データベース名：Musicrossing  
 
お使いのphpMyAdminに　Musicrossing.sql　をインポートをしていただきますと
データベース作成　→　テーブル作成　→　テストデータ登録　を全て一気に実行します。
